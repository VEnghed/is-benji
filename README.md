## Install

Install with [npm](https://www.npmjs.com/):

```sh
$ npm install is-benji --save
```

## Usage

Works with strings.

```js
const isBenji = require('is-benji)

console.log(isBenji('bongi)) //=> true
console.log(isBenji('plopp')) //=> false
```
